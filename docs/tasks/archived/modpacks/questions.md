# Questions & Concerns (Answered)

1. **Modpack Types & Overrides**: 
   - **Answer**: Respect the `overrides` field in `manifest.json`. For Modrinth, support both `overrides/` and `client-overrides/`.

2. **Mod Downloading**:
   - **Answer**: Use the current system for fetching from CurseForge.

3. **Instance Export**:
   - **Answer**: Allow the user to select which files to include/exclude via a folder/file navigation component.

4. **URL Support**:
   - **Answer**: Support Modrinth/CurseForge project URLs and allow version selection on the install page before proceeding.

5. **Resource Management**:
   - **Answer**: Yes, share downloaded mods between instances using the global cache.

6. **RAM Check**:
   - **Answer**: Check against physical total RAM.

7. **Icon Setting**:
   - **Answer**: Use the current system for instance icons (stored in DB).
